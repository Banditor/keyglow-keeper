# KeyGlow Keeper 1.0.0

## English

The first public release of KeyGlow Keeper keeps the keyboard backlight on for compatible Lenovo laptops without notifications, simulated key presses, administrator access, or Windows shell modifications.

Download the ZIP, extract it, and run `Install.cmd`. A standalone executable and SHA-256 checksums are also attached.

## עברית

הגרסה הציבורית הראשונה של KeyGlow Keeper שומרת את תאורת המקלדת דולקת במחשבי Lenovo תואמים, ללא התראות, הדמיית לחיצות, הרשאות מנהל או שינוי של ממשק Windows.

יש להוריד את קובץ ה-ZIP, לחלץ ולהפעיל את `Install.cmd`. מצורפים גם קובץ הפעלה עצמאי וערכי SHA-256 לאימות.
