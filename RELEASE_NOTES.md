# KeyGlow Keeper 1.0.1

## English

This patch restores the original yellow light-bulb tray icon when KeyGlow Keeper is active. The professional KeyGlow application icon remains embedded in the executable and Start menu shortcut.

Download the ZIP, extract it, and run `Install.cmd`. A standalone executable and SHA-256 checksums are also attached.

## עברית

גרסת תיקון זו מחזירה את סמל הנורה הצהובה כאשר KeyGlow Keeper פעיל. האייקון המקצועי של התוכנה נשאר משובץ בקובץ ההפעלה ובקיצור של תפריט התחל.

יש להוריד את קובץ ה-ZIP, לחלץ ולהפעיל את `Install.cmd`. מצורפים גם קובץ הפעלה עצמאי וערכי SHA-256 לאימות.
